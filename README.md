# Baseball Chat

A baseball statistics application with AI-powered player insights.

## Tech Stack

- **Frontend**: Nuxt 4 + Vue.js + TypeScript + TailwindCSS
- **Package Manager**: pnpm

## Quick Start

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Open browser
# http://localhost:3000
```

## Project Structure

```
baseball-chat/
├── pages/            # File-based routing
├── components/       # Vue components (auto-imported)
├── composables/      # Composables (auto-imported)
├── assets/           # CSS, fonts, images
├── public/           # Static files
├── .claude/          # Knowledge garden
└── nuxt.config.ts    # Nuxt configuration
```
